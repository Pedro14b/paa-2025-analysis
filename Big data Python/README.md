# PAA 2025 - Análise de Distribuição de Beneficiários

Projeto de análise de dados sobre a distribuição do Programa de Aquisição de Alimentos (PAA) para 2025.

## 📊 Sobre o Projeto

Este projeto realiza uma análise exploratória de dados (EDA) para investigar a distribuição heterogênea de beneficiários do PAA, identificando padrões, disparidades e propondo recomendações para otimização do programa.

## 🔍 Principais Descobertas

- 34 consumidores distribuídos em 7 categorias de entidades
- 79% da alocação concentrada em apenas 3 categorias
- Assimetria identificada com potencial para otimização
